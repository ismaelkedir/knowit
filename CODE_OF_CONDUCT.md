# Code of Conduct

## Our Standard

Participation in the Knowit project should be respectful, constructive, and professional.

Examples of expected behavior:

- giving actionable, respectful feedback
- assuming good intent while discussing tradeoffs
- welcoming contributors with different backgrounds and experience levels

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language or conduct
- publishing private information without permission
- disruptive or bad-faith behavior in project spaces

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this code of conduct.

## Reporting

Report conduct issues to the maintainers through GitHub issues only if the report can be made safely and appropriately in public. For sensitive reports, use the contact path in `SECURITY.md`.
